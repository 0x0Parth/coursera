# Paradise Nursery

Paradise Nursery is a React-based e-commerce application designed for plant enthusiasts. Users can browse a wide variety of houseplants categorized by their benefits, add them to a shopping cart, and manage their selections seamlessly.

## Features
- **Landing Page**: An inviting entrance with a "Get Started" button.
- **Product Listing**: 18 unique plants across 3 categories (Air Purifying, Aromatic, and Low Maintenance).
- **Shopping Cart**: Real-time updates using Redux for adding, removing, and adjusting plant quantities.
- **Responsive Design**: Styled with CSS for a clean user experience.

## Technologies Used
- React
- Redux Toolkit
- Vite
- CSS3
